## Data Import & Export

<Tabs>
  <Tab title="CSV Import">
    Bulk import customers, leads, and inventory via CSV:
    
    ```csv
    Customer Name,Email,Phone,Company,Address,City,State,Zip
    ABC Construction,contact@abc.com,555-0100,ABC Corp,123 Main St,Denver,CO,80202
    ```
    
    Features:
    - Automatic column mapping
    - Duplicate detection
    - Validation with error reporting
    - Batch processing for large files
  </Tab>

  <Tab title="Snowflake Sync">
    Connect to your data warehouse for automated imports:
    
    - Scheduled nightly imports
    - Custom column mapping per organization
    - Delta sync (only new/changed records)
    - Support for customers, inventory, transactions
  </Tab>

  <Tab title="API Integration">
    Use Quinn's REST API for real-time integration:
    
    ```javascript
    // Create customer via API
    const response = await fetch('/api/customers', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      credentials: 'include',
      body: JSON.stringify({
        customerName: 'New Customer',
        email: 'customer@example.com',
        phone: '+15551234567'
      })
    });
    ```
  </Tab>
</Tabs>
